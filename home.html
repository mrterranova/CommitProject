Commit to Github

Add what you want to this repo. Images, gifs, css, text... (Please be respectful). This is a free for all to learn how to commit on github.


GIT HUB INSTRUCTIONS

//this creates a new branch for you to use. Remember to name your branch something you will remember

>git checkout -b your_branch_name

//this means add everything to your branch

>git add .

//save a message

>git commit -m "put your message in here

//now switch back to master branch

>git checkout master

//now pull everything from the master branch

>git pull origin master

//now that all the master branch is loaded, switch back to the branch you named

>git checkout your_branch_name

//push your branch

>git push --set-upstream origin your_branch_name


Now go on the github repo located here: https://github.com/mrterranova/CommitProject

Click new pull request under <>Code

Since this repo has no checks on it, you can continue to commit your branch.



Working with Git and .ssh keys



If you try to clone or commit you may get an error like this.

              
                The authenticity of host 'github.com (192.30.252.129)' can't be established.
                RSA key fingerprint is 16:27:ac:a5:76:28:2d:36:63:1b:56:4d:eb:df:a6:48.
                Are you sure you want to continue connecting (yes/no)? 
                The authenticity of host 'github.com (192.30.252.128)' can't be established.
                RSA key fingerprint is 16:27:ac:a5:76:28:2d:36:63:1b:56:4d:eb:df:a6:48.
                Are you sure you want to continue connecting (yes/no)?

This means that you either don't have .ssh keys or the keys you have are wrong.
